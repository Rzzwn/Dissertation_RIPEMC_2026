Purpose
This repository contains the computational code for Layer 3 analysis of the dissertation Critical Discourse Analysis and Digital Reception of the 2025 Romanian and Polish Presidential Runoffs. The code processes YouTube comments to map semantic fields, sentiment distributions, and engagement patterns.

Data Availability
Raw YouTube comment data cannot be publicly shared due to YouTube API Terms of Service restrictions. Researchers may collect their own corpora using the provided scripts and documented API procedures. Code is available for replication and verification purposes.

License
This code is provided for academic replication purposes. Please cite the dissertation if used in research.
